<h3>That's a project I've developed by myself with the following functionalities:</h3>

- Play/Pause Video
- Increase/Decrease video speed 
- Theater mode
- Full Screen mode
- Volume settings

- Filter videos by category
- Select a new video to play
- Dark / Light mode
- Responsive for all devices
